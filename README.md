# 카붐로그 게시글 검색 기능 요구사항 
## 1. 검색 기준
사용자는 **검색어(keyword)**를 기준으로 아래 세 가지 방식 중 하나로 검색할 수 있어야 한다.
 - 제목(post_title) 기준 검색
 - 내용(post_content) 기준 검색
 - 제목 + 내용(post_title, post_content) 통합 검색 (제목에 가중치 2배 부여)

## 2. 동의어 처리
검색어는 다음과 같은 동의어 필터를 적용해 매칭 범위를 확장할 수 있어야 한다.
 - 예:
    - java ↔ 자바
    - ㅎㅇ ↔ 하이, hi, 안녕
    - programming ↔ 프로그래밍

## 3. 오타 허용 검색
fuzziness: AUTO를 통해 자동 오타 허용 검색을 지원하여 사용자 입력의 오류에 유연하게 대응할 수 있어야 한다.

## 4. 작성 기간 필터링
created_at 필드를 기반으로 한 작성일자 범위 필터 기능을 제공해야 한다.
- 예: 2024-01-01 ~ 2025-12-31 기간 내 게시글만 검색

## 5. 블로그 식별자(blog_fid) 필터링
특정 블로그에 속한 게시글만 조회하도록 blog_fid 조건 필터링이 가능해야 한다.

## 6. 페이지네이션
검색 결과는 size, from 값을 기반으로 페이지네이션이 가능해야 하며, 기본값은 다음과 같다:
 - size: 10개
 - from: 0번 인덱스부터 시작

## 7. 검색 전처리
검색 대상 텍스트는 다음 전처리 과정을 거쳐 분석된다.
 - nori_tokenizer 기반 토크나이징
 - 품사 필터 (nori_part_of_speech)
 - 표제어 추출 (nori_readingform)
 - 소문자화 (lowercase)
 - 불용어 제거 (stop)
 - 어간 추출 (stemmer)
 - 동의어 매핑 (caboomlog_synonyms)

## 8. HTML 및 마크다운 처리
본문 내 마크다운/HTML 기호는 무시되며, 의미 있는 텍스트 단위로만 검색되어야 한다.
